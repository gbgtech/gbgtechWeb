# What is this?

This is the website for gbgtech.co

## Required
* node
* npm
* running mongoDB server

## How to run
* `npm install`
* `npm seed` 
   * for seeding th mongoDB 
* `npm start`
    * Starts live reload on frontend and backend.


## Using
* react
* es6
* babel
* express
* mongodb
* webpack
