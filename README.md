## What is this?
This is the website for gbgtech.co

## Required
* node
* npm
* running mongoDB server

## How to run
* `npm install`
* `npm run seed` 
   * for seeding th mongoDB
* Copy /.env.template to .env and add keys(ask someone or fetch your own).
* `npm start`
    * Starts live reload on frontend and backend.
